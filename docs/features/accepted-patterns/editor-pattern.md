# Editor Pattern

## When To Use

- Focused, repeated actions such as attendance marking or other high-frequency edits.

## Why It Works

- It reduces friction and supports fast repetition with low cognitive load.

## What To Preserve

- Large tap targets.
- Immediate feedback.
- Search or filtering only when it directly speeds the task.

## What To Avoid

- Secondary controls competing with the main editing action.
- Layout shifts that slow repeated use.
